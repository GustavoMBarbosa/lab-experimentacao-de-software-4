# Dashboard Final – Laboratório de Experimentação de Software 4

Este diretório contém o **dashboard final** produzido no Power BI como parte da Sprint **Lab04S03**.

## Conteúdo do Arquivo PDF

O arquivo **DASHBOARD FINAL – Completo.pdf** reúne todas as visualizações criadas a partir do dataset analisado no Laboratório 3, incluindo:

### Caracterização do Dataset
- Distribuição do tamanho dos Pull Requests  
- Distribuição do tempo de análise  
- Número de revisões  
- Interações (participantes, comentários, reviews)

### Visualizações das Questões de Pesquisa (RQs)
- **RQ1:** Tamanho dos PRs × Feedback final  
- **RQ2:** Tempo de análise × Feedback final  
- **RQ3:** Descrição dos PRs × Feedback final  
- **RQ4:** Interações nos PRs × Feedback final  
- **RQ5:** Tamanho dos PRs × Número de revisões  
- **RQ6:** Tempo de análise × Número de revisões  
- **RQ7:** Descrição dos PRs × Número de revisões  
- **RQ8:** Interações nos PRs × Número de revisões  

## Ferramenta Utilizada
- **Power BI Desktop**  
  Exportação realizada via **Arquivo → Exportar → Exportar para PDF**.
